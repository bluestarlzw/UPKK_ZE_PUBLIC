## 可选参数
| 关键词 | 默认值|说明                                                                       |
|----------|-----|:----------------------------------------------------------------------------:|
| DoorCountDown  | 0 |手动设置倒计时 0 = 自动监听  -1 = 禁用自动监听  |
| Blocked  | false |设置成true则屏蔽该文本输出  |
| ClearTimer  | false |设置成true则在输出本文本时清空现有倒计时(可用于限时打死BOSS 打死BOSS后倒计时还在计时)  |
